The Java Swing Calendar Panel is a calendar component for your java swing applications. Here it is. Try it out!

The design is pretty simple. The calendar model just extends the java swing TableModel requiring a few extra items such as starting date and description.

Try this simple demonstration. The top panel is the calendar panel. Try dragging the starting and ending icons around to change the calendar times. Try changing the information in the bottom left panel and notice the model responds to the changes. (Those cool calendar popup buttons came from the JCalendarButton project) The bottom center panel shows how the calendar model appears in a swing JTable without any extra coding. The right panel is a thumbnail version of this component. Notice that it also responds to model changes and drag and drop actions.

To include JCalendarButton in your Maven projects, just add the following dependency to your Maven project:

<dependency>
   <groupId>${project.groupId}</groupId>
   <artifactId>${project.artifactId}</artifactId>
   <version>${project.version}</version>
</dependency>

This Calendar Panel is already configured as an OSGi bundle. To use it in your OSGi project, just include this in the manifest of the bundles that will be using Calendar Panel. Remember to install the Calendar Panel jar in your OSGi environment.

Import-Package: ${project.groupId}.calendar;resolution:=optional;version=${project.version}
